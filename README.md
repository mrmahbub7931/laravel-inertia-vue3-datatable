# Laravel 11 DataTable with Inertia.js and Vue 3

## Overview

This project is a simple DataTable implementation using Laravel 11 as the backend and Inertia.js with Vue 3 as the frontend. It demonstrates how to manage and display data efficiently in a modern web application.

## Prerequisites

- PHP 8.1 or higher
- Composer
- Node.js and npm
- A database (e.g., MySQL)

## Installation

### 1. Clone the Repository

```bash
git clone git@github.com:mrmahbub7931/laravel-inertia-vue3-datatable.git
cd laravel-inertia-vue3-datatable
```

### 2. Install Backend Dependencies

```bash
composer install
```

### 3. Set Up Environment
```bash
cp .env.example .env
```

### 4. Generate Application Key

```bash
php artisan key:generate
```

### 5. Install and Run NPM
```bash
npm install && npm run dev
```

### 6. Run Migrations
```bash
php artisan migrate
```
